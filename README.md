# Tic-Tac-Toe
A python implementation of the game Tic-Tac-Toe. Simple project to get my feet wet with python.
Requires 2 players.

# Usage
```
python tictactoe.py
```

# Controls
```
  Q | W | E             
 ----------            
  A | S | D             
 ----------              
  Z | X | C 
```
